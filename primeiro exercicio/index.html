O que é o HTML

voltar

HTML é uma linguagem de marcação para "dizer" para o navegador onde cada informação se encontra no site, ela apenas diz onde esta cada elemento e como eles devem se comportar de forma basica.

Essa formatação da página é feito de forma basica, sem estilização dos conteudos apresentados, para ser possivel deixar o site mais "bonito", seria preciso usar uma linguagem de estilização conhecida como css

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar e estruturar páginas na web. Ele utiliza uma série de elementos e tags para definir a apresentação e formatação do conteúdo, como textos, imagens, links e outros componentes. HTML é a base de qualquer site e é essencial para o desenvolvimento web.

Estudo das tags

voltar

As tag são como "blocos" de contrução do html, são essas tags que infromam como o html irá intermpretar o texto que esta dentro da tag, podendo ser tags para interpretar o texto como um titulo, parágrafo, lista, etc.

Temos varios tipos de tags para varias funcôes, essas tags podem ser encontradas na internet em documentaçôes de HTML

Estudo dos atributos

voltar

Os atributos são algumas modifiçãoes que se pode fazer nas tags, deixando elas masi completas ou com alguma modificação na sua funcionalidade, como por exemplo: mudar a cor de um texto, adicinar um link dentro do site, entre outras coisas

Esses atributos tambem podem ser encontrados na documentação de HTML dentro da internet
